## :information_source: How To Run

```sh
  # Clone this repository
  $ git clone https://github.com/felipefrm/lets-chat.git

  # Go into the repository
  cd lets-chat

  # install dependencies
  $ npm install

  # run the app
  $ npm run start

  # running on port 3000
```
